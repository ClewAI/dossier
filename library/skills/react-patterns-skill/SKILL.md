# React patterns skill

When creating or refactoring components:

- Split large components; extract hooks for reusable stateful logic.
- Verify lists, forms, and dialogs meet basic a11y (focus, labels, roles).
- Align with existing state management (Context, Redux, Zustand, etc.)—do not introduce a parallel pattern silently.

Use for feature UI, hook extraction, or performance tuning.
